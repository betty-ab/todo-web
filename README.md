#To-Do Web App

A responsive task management app built with HTML, CSS, and vanilla JavaScript.
It helps users organize tasks efficiently with features like drag-and-drop sorting, recurring tasks, and persistent storage.

🚀 Features

Add, edit, and delete tasks

Mark tasks as completed or pending

Search and filter tasks

Drag & drop task reordering (Sortable.js)

Recurring tasks (Daily, Weekly, Monthly)

Priority levels with color indicators

Dark mode toggle

Export tasks as a JSON file

Data saved using localStorage

 Tech Stack
 
-HTML
-CSS
-JavaScript
